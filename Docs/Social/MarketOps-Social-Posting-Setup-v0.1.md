# MarketOps Social Posting Setup v0.1

This guide supports manual posting preparation for Instagram, X, TikTok, and YouTube Shorts. It does not enable platform API posting.

## Current Mode

- Manual posting first
- Draft export only where needed
- No social auto-posting
- No API keys or tokens in this repo
- No credentials in chat or reports

## Instagram Manual Posting Start

1. Confirm the MarketOps account handle.
2. Store credentials in a password manager only.
3. Enable two-factor authentication.
4. Export reviewed image/video/caption drafts manually.
5. Confirm the caption says paper simulation or sample-data preview when discussing MarketOps outputs.

Later API readiness: Instagram publishing requires Meta/Instagram professional account setup, app review, permissions, and a separate security review.

## X Manual Posting Start

1. Confirm the MarketOps handle.
2. Store credentials in a password manager only.
3. Enable two-factor authentication.
4. Start with short build-in-public updates and dashboard-safe summaries.
5. Avoid trading instructions, guarantees, or private/internal details.

Later API readiness: X posting requires developer access, app configuration, scopes, and a separate review before any API posting code exists.

## TikTok Manual Posting Start

1. Confirm the MarketOps handle.
2. Store credentials in a password manager only.
3. Enable two-factor authentication.
4. Use reviewed short-form scripts and captions only.
5. Label market outputs as paper simulation, fake/sample money, or sample-data preview.

Later API readiness: TikTok requires a developer app, scopes, and platform review before API workflows are considered.

## YouTube Shorts Manual Posting Start

1. Confirm the MarketOps channel or handle.
2. Store credentials in a password manager only.
3. Enable two-factor authentication.
4. Upload reviewed Shorts manually.
5. Include not-financial-advice language where market research is discussed.

Later API readiness: YouTube uses Google/YouTube Data API with OAuth. That belongs in a later reviewed implementation, not this prep layer.

## Deferred Platforms

LinkedIn and Facebook are deferred until the writing and video quality is more polished.
