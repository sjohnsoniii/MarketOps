## The Production Loop

```
1. MarketOps scheduled refresh runs
   → npm run marketops:refresh
   
2. Paper-only bot state updates
   → cycle state, signals, risk, trades, equity
   
3. Public dashboard JSON updates
   → npm run public:trial-status
   → dashboard-bundle-public-v0.4.json regenerated
   
4. Public update manifest generated
   → npm run public:update-manifest
   → marketops-public-update-manifest-v0.1.json
   → Contains: generatedAt, refreshStatus, paperCycleId,
     universeProfile, bundle hash, git commits, publishStatus
   
5. Site data synced to sj3labs
   → sync-marketops-to-sj3labs.sh copies all files
   → Leak checks run (paths, API keys, prohibited claims)
   
6. Data-only git publish (allowlist-guarded)
   → pattern: data/marketops/*.json
   → If allowlisted-only changed: commit + push → Vercel deploys
   → If non-allowlisted changed: blocked + human-review report
   
7. Public dashboard renders fresh data
   → Fetches manifest first for versioned cache-busting
   → Uses manifest.generatedAt as ?v= parameter
   → Falls back to Date.now() if manifest unavailable
   → Shows "Data generated" and "Public update" timestamps
   
8. Scheduler report proves the chain completed
   → Reports publishStatus, Vercel trigger, all steps
```
