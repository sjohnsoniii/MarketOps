# Step 0 Definition of Done

Step 0 is COMPLETE when all of the following are true:

## Core Infrastructure
- paper runner operational
- autonomous office operational
- agent review layer operational
- scheduled tasks operational
- deterministic QA operational
- reports generated successfully

## Safety
- no live trading
- no broker APIs
- no live market data
- no social auto-posting
- no external signal transmission
- no production push without review

## Promotion Gate

Final Step 0 output should explicitly declare either:

STEP0_COMPLETE_READY_FOR_STEP1_FAKE_REALTIME_DATA_SETUP

OR

STEP0_INCOMPLETE
